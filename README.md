# Hotel_Management_System
A web-based Hotel Management System developed using Java, JSP, Servlets, JDBC, MySQL, HTML, and CSS. This project helps hotels manage reservations, customer details, room bookings, billing, and reports efficiently through an easy-to-use dashboard.
